# 🚀 OIBSIP Web Development Internship Projects

This repository contains all the tasks completed as part of the **Oasis Infobyte (OIBSIP) Web Development Internship**. Each task demonstrates fundamental concepts of web development including HTML, CSS, JavaScript, and basic backend structure.

---

## 📁 Project Structure

```
OIBSIP/
│
├── Task1_Calculator/
├── Task2_TodoApp/
├── Task3_TributePage/
├── Task4_LoginAuth/
```

---

## 🧮 Task 1: Calculator

A fully functional calculator built using:

* HTML
* CSS (Grid layout)
* JavaScript

### Features:

* Basic arithmetic operations (+, −, ×, ÷)
* Clear (C) button
* Responsive UI

---

## 📝 Task 2: To-Do App

A simple task management application.

### Features:

* Add tasks
* Delete tasks
* Interactive UI
* Helps improve productivity

---

## 🎤 Task 3: Tribute Page (BTS RM)

A tribute webpage dedicated to RM (Kim Nam-joon).

### Features:

* Biography and achievements

* Image integration
* Clean and readable design

## 🔐 Task 4: Login Authentication System

A basic login and registration system.

### Technologies Used:

* User login page

## 📌 How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/OIBSIP.git
   ```

2. Open any task folder.

3. Run the HTML file in your browser:

   ```
   index.html
   ```
